# 技术型销售核心能力

## 4个软实力

### 01 客户应用场景知识
- 把产品功能转化为客户能听懂的语言
- 了解客户每天要处理的工作、最耗时的环节、考核指标
- 预判障碍：客户说"预算不够"可能不是真没钱
- 准备好解决方案帮客户扫清障碍

### 02 行业知识梳理能力
- 建立行业知识库的方法：
  - 关注行业大佬的付费课程
  - 用自己的话"写"一遍（不是读）
  - 整理成自己的知识库
- 谈单时精准切入："最近很多同行都在头疼XX问题"

### 03 职场关系知识
- 明确决策链：
  - 使用者（技术部/业务部）：操作便捷性、功能实用性
  - 决策者（老板/采购）：成本、效率、ROI
  - 影响者（专家/意见领袖）：技术先进性、行业认可度
- 核心：价值互换，帮客户解决问题

### 04 工具型人际交往技能
- 精准倾听：让客户觉得被理解
- 适度共情：消除客户顾虑
- 高效跟进：定期传递价值（行业资讯、祝福、小礼物等）

## 核心观点
技术型销售的高阶技能 = 以客户为中心
- 懂场景 → 精准匹配需求
- 懂行业 → 建立深度信任
- 懂关系 → 打通决策链路
- 懂沟通 → 高效推进合作
